大学院で分子生物学・バイオインフォマティクスを専攻。  
メーカー研究職 -> ITメーカー 医療系データサイエンティスト -> 外資製薬エンジニア。  
副業ではバイオインフォマティクスの受託研究に関わり、論文執筆や解析パイプラインの構築を担当。

---

## 🧬 バイオインフォマティクス受託研究

研究者・企業向けに、バイオインフォマティクス分野の受託解析・開発支援を行っています。

**対応内容**

* 解析パイプラインの設計・構築（Python / Shellなど）
* NGSデータ解析（Metagenome / Microbiome解析 など）
* RNA構造予測・構造解析・モデリング
* 公開データ（SRA / GEO / ENA など）を用いた再解析
* 論文作成支援（Methods / Figure作成、Reproducibility担保）

**特徴**

* 研究背景を理解したうえでの解析設計
* 再現性・再利用性を重視したワークフロー構築
* クラウド・HPC環境への対応（AWS / Kubernetesなど）
